# Solution Design One-Pager Formatting Skill
Purpose
- Transform structured input into a clean, standardized one-pager.
- Ensure outputs are consistent, readable, and decision-ready.
---
## Scope
- This skill is only responsible for formatting and structuring output.
- Problem clarification and analysis are handled by the agent and other skills.
---
## When to use
- When the required information is already available or sufficiently clarified.
- When a structured summary for solution design input is needed.
---
## Input expectations
The input should already contain:
- A defined problem (or draft)
- Business objective (or partial)
- Scope elements
- Some indication of impacted areas
---
## Process

1. Organize the provided information into predefined sections  
2. Ensure clarity, consistency, and completeness  
3. Highlight missing sections if necessary  
4. Keep the output concise and structured  
---
## Output format
Always generate the following sections:
- Problem Statement  
- Business Objective  
- Scope (in/out)
